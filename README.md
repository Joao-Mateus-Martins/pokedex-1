## 🛠️ Como Rodar Localmente

### ✅ Pré-requisitos:

- Node.js + npm instalados

### 🚀 Passos:

```bash
# Clone o repositório
git clone <URL-do-seu-repositório.git>

cd pokedex

# Instale as dependências
npm install

# Rode a aplicação
npm start
```
